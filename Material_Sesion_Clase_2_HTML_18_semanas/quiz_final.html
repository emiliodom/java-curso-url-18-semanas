¿Qué aprendiste hoy?

10 preguntas sobre condicionales, ciclos, métodos y clases. Al terminar: Calificar → Exportar a PDF → subir al aula virtual.
1) ¿Cuál es la forma correcta de comparar dos textos (String) en Java?
✅ .equals() es el método correcto para comparar el contenido de dos Strings.
Es .equals(): == compara si son el mismo objeto en memoria, no el contenido.
2) En un switch, ¿qué pasa si olvidas el break dentro de un case?
✅ Correcto: sin break, la ejecución "cae" al siguiente caso.
Sigue ejecutando el siguiente case: es el "fallthrough", un error silencioso muy común.
3) ¿Cuántas veces imprime for (int i = 0; i < 5; i++) { System.out.println(i); }?
✅ Correcto: empieza en 0 y se detiene justo antes de llegar a 5.
Son 5 veces: i toma los valores 0,1,2,3,4 y se detiene cuando i < 5 es falso (i=5).
4) ¿Qué le falta a este ciclo para NO ser infinito? int i = 0; while (i < 10) { System.out.println(i); }
✅ Correcto: sin modificar i, la condición i < 10 siempre será verdadera.
Falta modificar i (por ejemplo i++) dentro del ciclo; si no, la condición nunca se vuelve falsa.
5) ¿Cuál de estas firmas de método es válida para uno que SÍ retorna un valor?
✅ Correcto: un método con tipo distinto de void debe incluir return con un valor de ese tipo.
La opción C: un método int exige un return que entregue un entero. La opción B da error "missing return statement".
6) ¿Cómo se llama al método saludar(String nombre) desde main, pasándole "Ana"?
✅ Correcto: se llama con paréntesis y el argumento dentro, como cualquier método.
Es saludar("Ana"); — paréntesis con el argumento adentro.
7) Con class Mascota { String nombre; }, ¿cómo creas un objeto y le asignas el nombre "Firulais"?
✅ Correcto: new crea el objeto y el punto accede al campo.
Es la opción A: new Mascota() crea el objeto y m.nombre = ... asigna su campo.
8) ¿Cuál es la diferencia principal entre una clase y un objeto?
✅ Correcto: puedes crear muchos objetos distintos a partir de la misma clase.
La clase es el molde (plano); el objeto es lo que se construye con new a partir de ese molde, con sus propios valores.
9) ¿Cuál método parece mejor nombrado según la convención de este curso (verbo/pregunta en camelCase)?
✅ Correcto: camelCase, empezando con minúscula.
Es calcularPromedio: los métodos (y variables) van en camelCase, no en mayúsculas ni con guiones bajos.
10) Tu compañero escribió un método esMayorDeEdad que retorna void en vez de boolean y ahora no puede usar el resultado en un if. ¿Cuál es el problema?
✅ Correcto: si necesitas usar el resultado en una condición, el método debe declarar y retornar boolean.
El método necesita ser boolean (con su return true/false) para poder usarse dentro de un if.
