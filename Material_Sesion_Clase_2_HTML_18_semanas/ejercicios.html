Ejercicio 1 — El semáforo Calentamiento

Objetivo: practicar if / else if / else.

  1. Declara String color = "amarillo";
  2. Si es "rojo" imprime "Alto". Si es "amarillo" imprime "Prepárate". Si es "verde" imprime "Avanza". Cualquier otro valor: "Color inválido".
  3. Usa .equals() para comparar Strings (no ==).
Pista
Para comparar texto en Java: color.equals("rojo"), no color == "rojo".
Solución sugerida 
public class Main {
  public static void main(String[] args) {
    String color = "amarillo";

    if (color.equals("rojo")) {
      System.out.println("Alto");
    } else if (color.equals("amarillo")) {
      System.out.println("Prepárate");
    } else if (color.equals("verde")) {
      System.out.println("Avanza");
    } else {
      System.out.println("Color inválido");
    }
  }
}

Ejercicio 2 — Días con switch Condicionales

Objetivo: practicar switch con break.

  1. Declara int mes = 7; (1 = enero … 12 = diciembre).
  2. Con switch, imprime el nombre del mes correspondiente.
  3. Agrega un default que imprima "Mes inválido".
  4. Quita un break a propósito, ejecuta y observa el "fallthrough". Vuelve a ponerlo.
Solución sugerida 
public class Main {
  public static void main(String[] args) {
    int mes = 7;
    String nombreMes;

    switch (mes) {
      case 7:
        nombreMes = "Julio";
        break;
      case 8:
        nombreMes = "Agosto";
        break;
      default:
        nombreMes = "Mes inválido";
    }

    System.out.println(nombreMes);
  }
}

Ejercicio 3 — Tabla de multiplicar Ciclo for

Objetivo: usar for para repetir un cálculo.

  1. Declara int numero = 7;.
  2. Con un for, imprime la tabla del 1 al 10: 7 x 1 = 7, 7 x 2 = 14, … 7 x 10 = 70.
Pista
for (int i = 1; i <= 10; i++) { ... numero * i ... }
Solución sugerida 
public class Main {
  public static void main(String[] args) {
    int numero = 7;

    for (int i = 1; i <= 10; i++) {
      System.out.println(numero + " x " + i + " = " + (numero * i));
    }
  }
}

Ejercicio 4 — Cuenta regresiva con while Ciclo while

Objetivo: usar while cuando no sabes exactamente cuántas vueltas serán.

  1. Declara int contador = 100; y una constante final int PASO = 15;.
  2. Con un while, resta PASO a contador en cada vuelta e imprime su valor, hasta que sea menor o igual a 0.
  3. ¿Cuántas vueltas dio el ciclo? Contesta en un comentario.
Solución sugerida 
public class Main {
  public static void main(String[] args) {
    int contador = 100;
    final int PASO = 15;

    while (contador > 0) {
      contador = contador - PASO;
      System.out.println("Contador: " + contador);
    }
    // Dio 7 vueltas (100, 85, 70... hasta -5)
  }
}

Ejercicio 5 — Método con parámetros Métodos

Objetivo: declarar un método con parámetros y valor de retorno.

  1. Crea el método static double calcularPromedio(double a, double b, double c) que retorne el promedio de los tres.
  2. Llámalo desde main con tres notas distintas e imprime el resultado.
  3. Llámalo una segunda vez con otras tres notas (demuestra que es reutilizable).
Solución sugerida 
public class Main {
  public static void main(String[] args) {
    double promedio1 = calcularPromedio(80, 90, 70);
    double promedio2 = calcularPromedio(60, 55, 65);

    System.out.println("Promedio 1: " + promedio1);
    System.out.println("Promedio 2: " + promedio2);
  }

  static double calcularPromedio(double a, double b, double c) {
    return (a + b + c) / 3;
  }
}

Ejercicio 6 — Método booleano Métodos + lógica

Objetivo: un método que retorna boolean, combinando lo visto en la Semana 1.

  1. Crea el método static boolean esPar(int numero) que retorne true si el número es par (usa %).
  2. Crea el método static boolean esAprobatorio(int nota) que retorne true si nota >= 61.
  3. Llama ambos métodos desde main con distintos valores e imprime los resultados.
Solución sugerida 
public class Main {
  public static void main(String[] args) {
    System.out.println(esPar(8));       // true
    System.out.println(esPar(7));       // false
    System.out.println(esAprobatorio(75)); // true
  }

  static boolean esPar(int numero) {
    return numero % 2 == 0;
  }

  static boolean esAprobatorio(int nota) {
    return nota >= 61;
  }
}

Ejercicio 7 — Mi primera clase: Mascota Clases y objetos

Objetivo: crear una clase simple con campos y un método, e instanciar dos objetos.

  1. Fuera de Main, crea class Mascota con campos String nombre y int edad.
  2. Agrégale el método void hacerSonido() que imprima nombre + " hace: ¡Guau!".
  3. En main, crea dos objetos Mascota con datos distintos y llama hacerSonido() en cada uno.
Pista
Recuerda: Mascota firulais = new Mascota(); y luego firulais.nombre = "Firulais";
Solución sugerida 
public class Main {
  public static void main(String[] args) {
    Mascota firulais = new Mascota();
    firulais.nombre = "Firulais";
    firulais.edad = 3;
    firulais.hacerSonido();

    Mascota michi = new Mascota();
    michi.nombre = "Michi";
    michi.edad = 1;
    michi.hacerSonido();
  }
}

class Mascota {
  String nombre;
  int edad;

  void hacerSonido() {
    System.out.println(nombre + " hace: ¡Guau!");
  }
}

Ejercicio 8 — Reto integrador: "Registro de beneficiarios" Integrador

Objetivo: combinar condicionales + ciclo + método + clase. Contexto del proyecto del curso.

  1. Crea class Beneficiario con campos String nombre e int edad.
  2. Crea el método static boolean puedeRecibirPrograma(int edad) que retorne true si edad >= 18.
  3. En main, crea un arreglo o simplemente 4 objetos Beneficiario distintos (puedes usar Beneficiario[] lista = new Beneficiario[4]; si ya viste arreglos, o simplemente 4 variables si no).
  4. Con un for (o repitiendo el bloque 4 veces si no usaste arreglo), imprime el nombre de cada uno y si puede recibir el programa (usa el método del paso 2 y un if).
Solución sugerida (versión simple, sin arreglos) 
public class Main {
  public static void main(String[] args) {
    Beneficiario b1 = new Beneficiario();
    b1.nombre = "Carlos";
    b1.edad = 20;

    Beneficiario b2 = new Beneficiario();
    b2.nombre = "Ana";
    b2.edad = 15;

    revisarBeneficiario(b1);
    revisarBeneficiario(b2);
  }

  static void revisarBeneficiario(Beneficiario b) {
    if (puedeRecibirPrograma(b.edad)) {
      System.out.println(b.nombre + ": SÍ puede recibir el programa");
    } else {
      System.out.println(b.nombre + ": NO puede recibir el programa (es menor de edad)");
    }
  }

  static boolean puedeRecibirPrograma(int edad) {
    return edad >= 18;
  }
}

class Beneficiario {
  String nombre;
  int edad;
}
Este patrón (clase + método que evalúa una regla de negocio) es exactamente lo que harás en Spring Boot desde la Semana 8.
