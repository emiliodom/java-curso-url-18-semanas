Semana 1 · Unidad 1a — Fundamentos de Java y tipos de datos · 11 de julio de 2026

Material de la Sesión 1 (formato 18 semanas · 70 min)

En esta sesión de 1 hora 10 minutos conoceremos el lenguaje Java y su ecosistema, declararemos variables, constantes y tipos primitivos, aplicaremos conversiones (casting) y practicaremos operadores aritméticos, lógicos y de comparación. No instalaremos nada: usaremos el compilador online de Educative.

1. Quiz de Presaberes 2. Abrir guía docente 3. Abrir ejercicios
Compilador Java (Educative)
https://www.educative.io/compilers/java
Sugerencia: abre el compilador en una pestaña y este material en otra. Todas las secciones tienen botón Exportar a PDF.

Aprendizajes esperados (hoy)

  • Comprende la estructura básica del lenguaje Java (clase + método main).
  • Declara variables, constantes y tipos primitivos.
  • Aplica conversiones (casting) entre tipos.
  • Aplica operadores y expresiones simples (aritméticos, lógicos y de comparación).

Ruta del estudiante (70 min)

  1. Quiz de Presaberes (5 min)
  2. Clase guiada: Java, variables y operadores (40 min)
  3. Challenge drag & drop + práctica (15 min)
  4. Quiz Final + ticket de salida (10 min)
Valores de la semana: resiliencia ante el error (el error de compilación es un maestro, no un fracaso) y pensamiento crítico (¿por qué Java exige tanta estructura?).
