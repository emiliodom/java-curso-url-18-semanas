Cheat Sheet · Semana 1

Referencia rápida — Java básico

Imprímela o tenla en una pestaña: es tu salvavidas durante ejercicios, challenge y la Guía 1.

1. Esqueleto de todo programa

public class Main {                      // contenedor
  public static void main(String[] args) { // punto de entrada
    System.out.println("Hola");            // instrucción + ;
  }                                        // cierra main
}                                          // cierra la clase
  • Toda instrucción termina en ;
  • Toda { que se abre, se cierra }
  • Mayúsculas/minúsculas importan: System ≠ system
  • println imprime y salta línea; print no salta

3. Nombres de variables (convención)

  • ✅ camelCase: anioNacimiento, promedioFinal
  • ✅ Constantes: MAYUSCULAS_CON_GUION: ANIO_ACTUAL
  • ❌ No empezar con número: 1nombre
  • ❌ Sin espacios ni tildes ni ñ: año nacimiento
  • ❌ No usar palabras reservadas: int, class…

6. Conversiones (casting)

// Implícito (chico → grande): automático
int x = 7;
double d = x;              // 7.0

// Explícito (grande → chico): TRUNCA
double precio = 9.99;
int p = (int) precio;      // 9 (no redondea)

// División entera vs decimal
7 / 2                      // 3
(double) 7 / 2             // 3.5

// Texto ↔ número
int n = Integer.parseInt("20");     // "20" → 20
double m = Double.parseDouble("1.5");
String s = String.valueOf(20);      // 20 → "20"
Truco del +: si hay texto en la mezcla, concatena. 1 + 2 + "3" → "33", pero "1" + 2 + 3 → "123" (se evalúa de izquierda a derecha).

7. Errores comunes y su traducción

El compilador dice…Significa…

incompatible typesIntentas guardar un dato en una cajita del tipo equivocado.

cannot assign a value to final variableIntentas cambiar una constante (final).
Protocolo ante un error: 1) respira, 2) lee la línea que indica el error, 3) corrige solo eso, 4) ejecuta de nuevo. El error es información, no fracaso.
